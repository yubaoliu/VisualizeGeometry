# About

Calculate and visualize Quaternion and Rotation matrix using Euler Angle.

# How to use

## Requrement

- Pangolin
- Eigen

## Compile

```sh
mkdir build
cd build
cmake ..
make
```

## RUN

- Input Yaw, pitch and roll use keyboard
    - yaw: Z-axis
    - pitch: Y-axis
    - roll: X-axis

- Output: Quaternion and Rotation matrix

- Visualize use Pangolin
